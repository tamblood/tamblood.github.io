Chefboard

Chefboard allows users to collect recipes from around the web and store them in a place that is visually rich and easy to search, a 'pinterest for recipes.' This final project for Dev Bootcamp is a single page app built on Rails and AngularJS and deployed on Heroku. Team members: Meg Sandman, Charlie Dibble, Andrea Braunstein, Tam Blood.

Chefboard on Heroku »

Restaurant Peek

Restaurant Peek allows user to choose favorite restaurants from which to receive a text message with the daily special. Restaurant Peek is a single page app written using Sinatra, Ruby, JavaScript and AJAX and deployed on Heroku. Nokogiri gem used to scrape websites, Twilio API used to send text messages.
